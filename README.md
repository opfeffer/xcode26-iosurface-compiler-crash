> [!NOTE]
> Filed Feedback with Apple on 6/24/2025 under **FB18354920**.

Reproduces with
- Xcode 26.0 beta 1 (17A5241e)
- Xcode 26.0 beta 2 (17A5241o)

Other system details:
- Running macOS Sequoia 15.5 (24F74) 

### Steps to reproduce

1. Download/Clone project
1. Build `IOSurfaceCrash` target with Xcode 26.0 beta 1 or beta 2
1. Observe build failures with the failure information seen below.

